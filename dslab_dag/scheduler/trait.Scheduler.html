Expand description

Trait for implementing DAG scheduling algorithms.

Includes callback methods which can return one or multiple actions corresponding to decisions made by the scheduler (assign task to resource, transfer data item between resources, etc).

Useful for implementing dynamic scheduling algorithms. For static algorithms just return Vec::new().
