pub struct DAGRunner { /* private fields */ }
Expand description

Manages the execution of a DAG on a specified set of computing resources.

Invokes the supplied scheduler and executes the actions returned by the scheduler. Tracks and updates the states of DAG tasks, data items and data transfers. Supports collection of DAG execution log.
